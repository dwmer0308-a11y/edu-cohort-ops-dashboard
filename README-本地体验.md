# 营期转化监控看板：本地体验说明

## 1. 打开方式

先解压这个文件夹，然后在终端进入解压后的目录。

运行：

```bash
python3 server.py
```

看到下面这行后，就说明启动成功：

```text
营期转化监控看板已启动：http://127.0.0.1:8765
```

然后用浏览器打开：

```text
http://127.0.0.1:8765
```

## 2. 如果提示缺少 openpyxl

运行：

```bash
pip3 install openpyxl
```

然后再执行：

```bash
python3 server.py
```

## 3. 数据保存在哪里

本地配置保存在：

```text
data/config.json
```

看板里的渠道、子渠道、老师、日历目标、R值模板、建期配置，都会保存在这个 JSON 文件里。

## 4. 体验顺序建议

1. 先进入「配置中心」，确认渠道、子渠道、老师。
2. 进入「日历目标」，填写每天每个子渠道的 Leads 目标。
3. 进入「R值模板」，填写 D4-D14 的目标 R值。
4. 进入「建期中心」，创建营期。
5. 回到「总览」上传 Excel，查看目标和实际的对比。
