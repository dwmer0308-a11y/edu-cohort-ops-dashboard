# 已发生营期导入字段说明

当前支持导入 `JSON`、`CSV`，局域网/服务版额外支持 `XLSX / XLSM / XLS`。

## 必须字段

| 含义 | 推荐字段名 | 兼容字段名 |
| --- | --- | --- |
| 营期名称 | `name` | `campaignName`、`营期`、`营期名`、`营期名称` |
| 实际 Leads | `actualLeads` | `leads`、`leads数`、`Leads`、`实际Leads`、`实际leads` |
| 实际 GMV | `actualGmv` | `gmv`、`GMV`、`实际GMV`、`实际gmv` |

## 可选字段

| 含义 | 推荐字段名 | 兼容字段名 |
| --- | --- | --- |
| 消耗 | `spend` | `消耗` |
| 正价课学员数 | `fullPriceStudents` | `正价课学员数`、`正价学员数` |
| 分类/子渠道 | `categories` | `分类` |
| 来源备注 | `source` | - |

## 当前用户上传文件检查结果

`营期_渠道数据统计_张良（实时）_2026-05-19T13_18_05.119518+08_00.csv` 已完成识别。

系统会按以下规则处理：

| 行类型 | 识别规则 | 用途 |
| --- | --- | --- |
| 营期聚合 | `工作室 = 营期汇总` 且 `分类 = -` | 作为该营期的实际 Leads、GMV、R值、转化率、ROI |
| 工作室汇总 | `分类 = 工作室汇总` | 存为该营期的渠道/工作室汇总明细 |
| 子渠道明细 | 其他分类，例如 `BD1-图书`、`BD1-0元`、`私域`、`其他` | 存为该营期的子渠道实际明细 |

实际导入会按 `营期` 字段自动匹配在线创建的规划营期；如果没有匹配到规划营期，会在营期监控里标记为 `未匹配规划`。
